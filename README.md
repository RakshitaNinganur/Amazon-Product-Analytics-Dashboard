# Amazon Product Analytics Dashboard

## Project Overview
An interactive Power BI dashboard built to analyze Amazon product data, including product categories, prices, discounts, ratings, and customer reviews.

## Tools Used
- Power BI
- Excel
- SQL
- Power Query
- DAX

## Dashboard Pages
### Executive Dashboard
- Total Products
- Average Rating
- Average Discount
- Product Distribution
- Top Categories

### Product Details
- Product Filters
- Product Information
- Top 10 Highest Discount Products

### Business Insights
- Highest Discount
- Highest Rating
- Total Reviews
- Average Price Difference
- Category Analysis

## Dataset
The dataset contains product details, prices, discounts, ratings, and review counts.

## Skills Demonstrated
- Data Cleaning
- Data Modeling
- DAX
- Dashboard Design
- Data Visualization
- Business Intelligence
